# Myntra-dataset-samples

<h2>1001개 Myntra 제품의 샘플 データセット</h2>

![Myntra dataset header](https://github.com/luminati-io/myntra-dataset-sample/blob/main/Myntra-datasets.png)

1000개 이상의 레코드로 구성된 Myntra データセット 샘플입니다. データセット는 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>Myntra データセット에 포함된 일부 데이터 포인트:</h2>

* ```url```: 제품의 URL 또는 링크
* ```product_id```: 제품의 고유 식별자
* ```title```: 제품의 제목 또는 이름
* ```product_description```: 제품 설명
* ```rating```: 제품의 평균 평점
* ```ratings_count```: 제품의 평점 또는 리뷰 수
* ```initial_price```: 제품의 초기/원래 가격
* ```discount```: 제품에 제공되는 할인
* ```final_price```: 할인 후 제품의 최종 가격
* ```currency```: 가격 책정에 사용된 통화

그 외에도 훨씬 더 많은 항목이 있습니다.

이는 "Myntra products (public data)" データセット에서 파생된 샘플 서브셋이며,
해당 データセット에는 <b>190만 개 이상의 레코드</b>가 포함되어 있습니다.

사용 가능한 データセット 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, 또는 Parquet. 선택적으로 파일을 .gz로 압축할 수 있습니다</b>.

データセット 제공 방식 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 주기: <b>Once, Daily, Weekly, Monthly, Quarterly, 또는 Custom basis</b>.

추출된 데이터 포인트에 추가로 제공 가능한 데이터 보강: <b>요청에 따라 제공됩니다.</b>

<b>[전체 Myntra データセット 받기](https://brightdata.co.kr/products/datasets/myntra)</b>.

<h2>Myntra データセット의 활용 사례는 무엇입니까?</h2>

<h3>1. Pricing Strategy and Optimization</h3>
Myntra에서 확인할 수 있는 경쟁사의 유사 제품 및 카테고리를 분석하여 동적 가격 책정 모델로 견고한 가격 전략을 수립합니다.

<h3>2. Product Inventory Gap Analysis</h3>
제품 재고의 격차를 파악하고 소비자 수요 트렌드를 이해하여 제품 구성을 최적화합니다.

<h3>3. Market Strategy Optimization</h3>
Myntra データセット 내 트렌드와 고객 선호도를 분석하여 시장 전략을 최적화합니다.
